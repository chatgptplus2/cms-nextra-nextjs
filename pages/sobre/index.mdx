# Sobre este CMS

Este projeto foi feito com:

- **Next.js** para renderização e rotas.
- **Nextra** para transformar `.md`/`.mdx` em páginas.
- **Sem banco de dados**: todo o conteúdo fica em arquivos Markdown.

## Fluxo editorial

1. Crie um arquivo em `pages/blog/`.
2. Adicione frontmatter e conteúdo.
3. Inclua o slug no `pages/blog/_meta.json`.

Pronto: o artigo terá sua URL automaticamente em `/blog/seu-slug`.
